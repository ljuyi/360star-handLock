js中的基本类型和引用类型

2016-11-30 • 标签: js

js中的数据类型有以下几种：

number boolean undefined object function string null

js中的数据类型主要有两种：基本类型 和 引用类型

基本类型

number boolean string undefined null

基本类型存储在栈内存中,基本类型的复制会在栈内存中重新开辟一个空间存储这个新的变量，也就是说，改变其中一个变量的值，另一个值不会改变

引用类型

object function

引用类型存储在堆内存中，但是在栈内存中会开辟一个空间存储这个引用类型在堆内存的地址，相当于在栈内存中存了一个指针指向堆内存中的该引用类型
如果发生引用类型的复制，则相当于在栈内存中又开辟一块空间存储一个指针，指向相同的堆内存中的引用类型，也就是说，修改其中一个引用类型的值，则另一个的值也会改变

需要注意的是，基本类型和引用类型还有一个区别：基本类型是没有属性和方法的，只有引用类型有

基本类型如果定义了属性或者方法，则会立即销毁，所以执行下一句的时候，string的index属性其实已经不存在了
